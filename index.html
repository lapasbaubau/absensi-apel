Halaman QR Code Absensi

Gunakan halaman ini untuk melakukan absensi dengan memindai QR Code

Petunjuk Penggunaan

  1. Pastikan Anda terhubung ke jaringan internet
  2. Buka aplikasi pemindai QR Code di perangkat Anda
  3. Arahkan kamera ke QR Code yang ditampilkan di sebelah kanan
  4. Tunggu hingga QR Code terbaca dan mengarahkan ke halaman absensi
  5. Ikuti instruksi yang muncul di halaman absensi
Catatan Penting: Jika QR Code tidak muncul, klik tombol "Buka Halaman QR Code" di bagian bawah untuk mengakses halaman absensi secara langsung.

Gunakan tombol di bawah untuk alternatif lainnya.
